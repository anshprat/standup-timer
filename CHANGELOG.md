# Changelog

All notable changes to the Standup Timer Chrome Extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2025-12-05

### Added
- Add automatic changelog generation to package.sh

### Fixed
-  preserve pause state when switching participants and fix total time calculation


## [1.0.0] - 2025-12-05

### Added
- Initial release of Standup Timer Chrome Extension
- Embedded timer overlay for team standup meetings
- Participant rotation with previous/next navigation
- Countdown and countup timer modes
- Time allocation per participant
- Pause/resume functionality
- Minimizable timer widget
- Summary report at meeting end with individual participant times
- Persistent settings storage
- Timer position customization
